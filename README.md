# Corrugated CRM

Frappe app that adds corrugated-industry-specific fields to CRM Lead and CRM Deal, a printable Spec Card, and an on-screen audit trail widget.

## Features
- **Corrugated Specs section** on CRM Lead and CRM Deal: Box Style, Flute Type, Board Grade, L×W×D dimensions, Annual Quantity
- **Print Specifications**: # Colors, Print Method, Coating
- **CAD & Artwork file attachments** on every Lead and Deal
- **Corrugated Spec Card** print format (PDF-ready, 1-page)
- **New Estimate** button to create a Corrugated Estimate directly from a Lead or Deal
- **Change History** audit trail widget on every form (powered by Frappe track_changes)
